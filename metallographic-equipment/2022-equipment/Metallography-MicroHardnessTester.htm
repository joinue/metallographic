ALPHA-MHT-1000Z Microhardness Tester

ALPHA-MHT-1000Z is an auto-turret Micro Vickers hardness testing instrument for Vickers and Knoop indenters (Part No. ALPHA-MHT-1000Z). Benefits include variable testing forces (10, 25, 50, 100, 200, 300, 500, 1000 gram load), 10X and 40X objectives, automatic loading, dwell and unloading along with an auto-turret head to prevent damage to the indenter. The ALPHA-MHT-1000Z can save and export data to either a printer or a PC via a RS232 interface cable.

ALPHA-MHT-2000Z2 Microhardness Tester

ALPHA-MHT-2000Z is an auto turret touch screen precision Micro Vickers hardness testing instrument for Vickers and Knoop indenters (Part No. ALPHA-MHT-2000Z2). Features include variable testing forces (10, 25, 50, 100, 200, 300, 500, 1000 and 2000 gram load), 10X and 40X objectives, a large touch screen for displaying measuring methods, testing force, indentation length, hardness value, dwell time and the number of measurements.

ALPHA-MHT-AUTO Fully Automated Microhardness Tester

ALPHA-MHT-AUTO is a fully automated Micro Vickers hardness testing instrument for Vickers and Knoop indenters (Part No. ALPHA-MHT-AUTO). Features include variable testing forces (10, 25, 50, 100, 200, 300, 500, and 1000 gram load), 10X. 20X and 40X objectives, touch screen control, CCD image acquisition system, motorized X-Y table, automatic focusing and automatic measurement. The hardness-depth data output is in Microsoft Word or Excel.
