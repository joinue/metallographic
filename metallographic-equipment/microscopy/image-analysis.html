Metallographic Image Analysis Software

Metallographic Image Analysis Modules

Our Metallographic Image Analysis software offers an integrated solution for studying and documenting microstructures with advanced image processing. Built to meet ASTM standards, it supports phase quantification, inclusion analysis, grain size measurement, nodularity assessment, porosity evaluation, decarburization detection, and coating thickness measurement. High-resolution images are captured via a digital microscope camera and imported for detailed analysis and storage.

Materials Plus Image Analysis

Features

  • Comprehensive Analysis: Offers a wide range of metallographic analysis tools, including phase quantification, grain size measurement, and coating thickness evaluation.
  • Automated Processing: Fast and accurate image processing techniques for consistent and reliable results.
  • ASTM Standards Compliance: Designed to meet ASTM standards for various metallographic measurements.
  • Integrated Reporting: Generate detailed reports directly from the analysis software for easy documentation.

Features
